# Задание 1
## Задача 4
### Текст задачи:
Время. 

Создайте сущность Время, которое будет описывать текущее время суток в 24-х часовом 
формате. Время описывается числом секунд, прошедшим с начала суток. Время может быть 
приведено к текстовой форме следующего формата: “ЧЧ:ММ:СС”. Например,если время задано как 
12000, то текстовая форма будет “3:20:00”. Если общее время превышает 24 часа, то отображаться 
в текстовом виде должно только то время, которое прошло с начала последних суток, например 
91800, это не 25:30:00, а 1:30:00. 

Необходимо создать и вывести на экран текстовую форму для следующих вариантов времени: 
- 10 секунд 
- 10000 секунд 
- 100000 секунд

### Алгоритм решения:
1. Проектирование класса
- Создаем класс Task1_4 с одним приватным полем seconds

3. Алгоритм преобразования секунд в формат ЧЧ:ММ:СС

Шаг 1: Нормализация секунд
- totalSeconds = seconds % (24 * 3600)
- 24 * 3600 = 86400 секунд в сутках
- Операция % (остаток от деления) отбрасывает полные сутки
- Получаем количество секунд с начала последних суток (от 0 до 86399)

Шаг 2: Вычисление часов
- hours = totalSeconds / 3600
- Делим общее количество секунд на 3600 (секунд в часе)
- Получаем целое количество часов (от 0 до 23)

Шаг 3: Вычисление минут
- minutes = (totalSeconds % 3600) / 60
- totalSeconds % 3600 — секунды, оставшиеся после выделения часов
- Деление на 60 преобразует эти секунды в минуты

Шаг 4: Вычисление секунд
- secs = totalSeconds % 60
- Остаток от деления на 60 даёт секунды текущей минуты

4. Форматирование результата
- String.format("%d:%02d:%02d", hours, minutes, secs)
- %d — целое число для часов
- %02d — двузначное число с ведущим нулём для минут и секунд
